# Account Manager
This project wants to help you save your accounts credentials on your PC, so you can easily logIn to all sites

## Infos
>- [Python 3.11](https://docs.python.org/3/)
>- [Visual Studio Code](https://code.visualstudio.com/)

**Library**
>- [Python Cryptography](https://cryptography.io/en/latest/)
>- [Python PyperClip](https://pyperclip.readthedocs.io/en/latest/)

**Install library command (Windows users)**
```
pip install cryptography
pip install pyperclip
```
